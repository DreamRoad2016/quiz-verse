竞猜宇宙

在限定次数内，根据多维属性反馈猜出隐藏人物——灵感来自经典猜谜范式， 未来将覆盖历史、影视、体育与更多题材；对战与房间玩法正在路上。

大类题库 标签筛选 服务端比对 绿 / 黄 / 灰反馈

🎯 怎么玩

🎲

随机答案

系统从当前题库随机选定一名隐藏人物，你不知道是谁。

⌨️

输入猜测

输入另一名人物的 ID 或姓名，联想提示帮你快速锁定。

🧩

属性反馈

每猜一次，多列属性用颜色告诉你离答案更近还是更远。

🏆

猜中为止

在次数用尽前命中答案即胜利；未来将支持好友同题对战。

单人经典：表格反馈中 绿色为一致， 黄色为集合部分重合或数值接近， 灰色为不一致；数值列可能带 ↑↓。

题库结构：按大类组织，同一局只用一个大类内的条目；大类内标签缩小范围（开发中）。
